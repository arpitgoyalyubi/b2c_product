How attribution works — decision tree
Scenario Attribution method Invitee action Coverage
Tapped referral link → installed Branch deferred deep link None — auto-captured ~70% of Android, ~60% of iOS
iOS ATT declined — link clicked No attribution data Manual code field ~25–30% of iOS
Searched App Store manually No attribution data Manual code field Edge case
Signed up on web (aspero.in) URL query param `?ref=ARPIT2026` None — auto-captured All web signups
Signed up late (code not entered at signup) Grace window Enter code within 7 days Captures late-stage code share
!
Never block signup for missing/invalid code If attribution fails and the user doesn't enter a code, signup proceeds normally. The referral is simply unattributed. Blocking or alerting heavily on a missing code would cause more drop-off than the unattributed referral is worth.
✓
7-day grace window Users who signed up without a code can enter one within 7 days of signup. This covers the common scenario where someone installs, signs up, then contacts their friend to ask for the code.
