Cross-Category Overlaps

⚠ Some issues appear across multiple categories — counted once in totals above

Marital status selector bug (~15–18 unique tickets): Split across App/Bug (~10) and Personal Details (~8) depending on how users described it. Same root cause — UI element unresponsive on certain devices.

Regulatory Check / KRA failure (~80+ unique tickets): Shows up in App/Bug, Demat, and Bank Account — users hit it at different onboarding stages. Single root cause in KRA API / compliance check flow.

KYC stuck / verification limbo (~20 tickets): Bleeds across PAN Issue, Demat, and KYC / Onboarding. Users who don't know which step failed raise it under whichever category the form last showed them.
